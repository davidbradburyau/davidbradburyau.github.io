Skills Required

The skills required for implementing the business technology solution include the following;
-Ability to physically install and correctly wire the main network and all subsystems. This would require a knowledge base to understand why things need to be connected in a certain way.
-Ability to configure networks and network products.
- Ability to set up and maintain software on employees’ computers including cloud access and remote access for working from home.
- Ability to set up and configure IP audio visual, IP security products, IP intercom systems and IP automation systems.
-A good knowledge of internet security and the ability to implement tools whether it be software or hardware related.
